# Assignment 1 — ES6 Refactor Pack

## How to Run

Run the legacy version:

node legacy.js

Run the modern version:

node modern.js

## What This Program Does

The legacy code filters admin users from a list.

The modern version refactors the code using ES6 features.

## ES6 Features Used

- const / let
- Destructuring
- Template literals
- Array methods (filter, reduce, find)
- Modules (functions separated into utils.js)

## What I Learned

- How to refactor JavaScript using modern ES6 syntax
- How to split code into modules
- How to use array methods like filter and reduce

## Challenges

Understanding module imports and debugging Node errors.
